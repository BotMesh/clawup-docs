# Quick Start

1. Sign in with Google/GitHub/Wallet.
2. Recharge balance in **Billing**.
3. Create your first Claw:
   - Pick a model using the searchable model picker (type to filter by name or provider).
   - Enter your API Key.
   - Select channels (Telegram, Feishu, etc.) — advanced options like Claw Type and Restore Source are collapsed by default.
4. (Node admin) Optional: in **Settings -> Users**, configure deploy-account allowlist for specific users.
5. Open **Apps -> Marketplace**, select your Claw, and add needed apps.
6. Configure channels and complete pairing in chat.
7. Test chat and monitor runtime status.

If you use Telegram, verify and recover webhook sync from **Apps -> Installed -> Resync Webhook**.
